# ChefNeYapsam

Malzemeler girildikten sonra yapilabilecek yemekleri oneren basit bir yemek sitesi.

## Nasil calisir?

- Kullanici malzemeleri kutuya yazar.
- Sistem 50 tariflik yerel yemek arsivini tarar.
- Girilen malzemelere gore en uyumlu tarifleri listeler.
- Ayrica sitede 100 farkli malzeme tanimlidir.

## Baslatma

1. `index.html` dosyasini tarayicida ac.
2. Malzemeleri yaz.
3. `Yemek Oner` butonuna bas.

Bu surum tamamen yerel calisir ve API anahtari gerekmez.

## Sonraki adim

Istersen bunu bir sonraki adimda:

- Tarif kartlarina gorsel ekleyebiliriz.
- Kategori filtresi ekleyebiliriz.
- Favorilere ekleme ozelligi koyabiliriz.
- Tarif detay sayfasi ekleyebiliriz.
- Mobil uygulama gibi gorunen daha profesyonel bir tasarima cevirebiliriz.
